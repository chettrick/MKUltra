MKUltra

Group Members:

Iain Emslie

Project Topic:

We would like to create a software musical synthesizer for our course project. Ideally this will be in the form of a desktop GUI-style interactive application.

Initial Survey:

While this has been done before in many different ways it would be a valuable learning exercise to investigate all of the mathematics that goes into software signal processing, the process of creating an application and the multimedia requirements discussed in class (MIDI, wav files etc).

Deliverables:

The deliverables will be in the form of a project report and diary, the code for the project and a demonstration of the software.
