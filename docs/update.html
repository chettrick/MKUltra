Project Topic:

We would like to create a software musical synthesizer for our course project. Ideally this will be in the form of a desktop GUI-style interactive application.

Technical Challenges Solved:

  • Built iterations of software synthesizers, focusing on audio waveform generation.
  • Began developing object oriented modular architecture as a foundation for further expansion.
  • Implemented Envelope, Oscillator, Instrument, and Mixer module classes.
  • Integrated PortAudio audio streaming library into a Sound class, which is usable by all modules.
  • Implemented a real-time MIDI interface to external MIDI keyboard controllers.
  • Integrate CMake project build management, which works cross-platform.

Technical Challenges Remaining:

  • Implement GUI features using wxWidgets or similar library
  • Further develop the existing midi and audio systems so that input/output interfaces can be chosen.
  • Further refine the existing modules and create new ones for filters and effects (reverb, delay) etc.
  • Make the code something of a library that can be used across platforms (OSX, Windows, Linux etc.).
  • Transition real-time computation of audio sample values to a precomputed wavetable approach.
